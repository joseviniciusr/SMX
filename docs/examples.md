# Examples

SMX ships runnable scripts and a notebook under `examples/`:

- `examples/quickstart.py` - end-to-end pipeline using synthetic data
- `examples/quickstart.ipynb` - notebook version of the quickstart
- `examples/generate_gallery_images.py` - regenerates the image assets used in docs

To run the quickstart script:

```bash
python examples/quickstart.py
```
